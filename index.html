MY PROJECTS

GameShot, Instagram for Gamers

GameShot was built using Rails. I implemented Paperclip with AWS for image upload capabilities, and Ransack for user database search to 'Follow' people and see their posts. I used Devise for user authentication and creation.

Brick Breaker

Brick Breaker is built using Javascript. The game is a canvas element, allowing me to move the ball through animation loop and interations bewteen the objects. I researched collision detection and created my own detection logic to properly bounce the ball off the blocks and game window wall.
